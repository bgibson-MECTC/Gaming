A simple, interactive math quiz game to test your arithmetic skills!

## Features

- **Timed Questions**: Answer each question within 20 seconds
- **Multiple Choice**: Choose from 4 possible answers
- **Score Tracking**: Earn points for correct answers
- **Instant Feedback**: See explanations for each answer
- **Skip Option**: Skip difficult questions if needed
- **Responsive Design**: Works on desktop and mobile devices

## How to Play

1. Open `index.html` in a web browser
2. Enter your name
3. Click "Start Quiz"
4. Select the correct answer for each question
5. Try to answer before the timer runs out!
6. Review your final score and play again

## File Structure

- `index.html` - Main game page
- `styles.css` - Game styling and animations
- `game.js` - Game logic and functionality
- `questions.json` - Quiz questions and answers
- `README.md` - This file

## Customization

You can customize the game by:

- Editing `questions.json` to add more math questions
- Changing the timer duration in `game.js` (default: 20 seconds)
- Modifying point values for each question
- Updating the color scheme in `styles.css`
